HANGMAN

This project involved enhancing a very basic existing Hangman game file.

The Key enhancements included:

  • Creating a user greeting function.
  • A random word selector from a saved file.
  • A menu for selecting from multiple word file.
  • Difficulty variations in lives or word difficulty.
  • An enhanced game over screen.

The key deliverables included:

  • A system change request documenting the key changes.
  • An analysis and design document detailing the enhancements.
  • A comprehensive test plan, including unit and integration tests.
  • A user manual for the updated version of the game
