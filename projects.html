HANGMAN

This project involved using Python3 to enhancing a very basic existing Hangman game file.

The Key enhancements included:

  • Creating a user greeting function.
  • A random word selector from a saved file.
  • A menu for selecting from multiple word files.
  • Difficulty variations for lives or word difficulty.
  • An enhanced game over screen.

The key deliverables included:

  • A system change request documenting the key changes.
  • An analysis and design document detailing the enhancements.
  • A comprehensive test plan, including unit and integration tests.
  • A user manual for the updated version of the game

THE ISLAND

This project involved creating a vintage text-based adventure game set in a 2D grid world layout using Python3 and an object orientated design.

The game had to include:

  • At least 10 locations, explorable and navigatable to using cardinal directions.
  • At least 3 non player characters that can be interacted with.
  • At least 10 interactive items, some being able to be picked up and stored in an inventory.
  • Conditional events that reaquire other actions to be taken or items to be used.
  • A file-based system to track visited locations using random access file operations
  • A goal to complete the game or potentially lose the game through incorrect actions.
  • Conditional events that reaquire other actions to be taken or items to be used.

I also chose to include:

  • Specialisations giving different perks.
  • An energy and hydration system to make players more deliberate with their movements.
